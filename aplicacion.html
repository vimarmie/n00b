Aplicación

Pero ahora, para poner a prueba algo de lo aprendido, se les solicita:

  1. Hacer unas pruebas de modificación del estilo.css del sitio web de esta clase
  2. Subir el sitio web modificado en un repositorio de GitHub
  3. Aprovechar las GitHub Pages para publicar lo del respositorio como sitio web.
  4. Enviar la URL de su sitio web a profesor@faco.cl, con copia a fcortez@uc.cl. Como asunto, favor escribir: "Aplicación: Apellido + Apellido"

Recomendación

Para la próxima clase, se les recomienda traer impresa una "cheat sheet" de:

  • HTML 5
  • CSS 3

En caso les interese profundizar en procesos de producción web, para los próximos semestres se les recomienda:

    • Usabilidad y Nuevos Medios, 10 créditos, cód. DNO046
